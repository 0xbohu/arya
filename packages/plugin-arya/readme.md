# Starknet Arya Plugin

## Overview

Arya is a generative agent can interact with Starknet ecosystem

## Build Arya

```
cd arya/packages/plugin-arya
pnpm build
```

## Start the agent

```
cd arya/packages/plugin-arya
pnpm start --character="characters/arya.character.json"
```

## Interacting with the agent on browser

```
cd arya
pnpm start:client
```
